О проекте

Это моё клиент-серверное одностраничное веб-приложение для управления собственными задачами. Каждая задача состоит непосредственно из строки, в которой сформулирована цель задачи, и флага, хранящего информацию о статусе выполнения задачи. Все задачи отображаются на странице в виде единого списка, причём редактирование задач и изменение их статуса происходит тоже внутри общего списка. Вся информация о каждой задаче сохраняется в базе данных на бэкенде.

Teхнологии

Для создания приложения использовались React, Django и Django Rest Framework.
Рабочую версию приложения можно посмотреть здесь.
Код можно посмотреть здесь.

Features

  • Пользователь может изменить статус выполнения задачи. То есть возможно отметить любую задачу как исполненную или наоборот (при этом эти два типа задач визуально отличаются).
  • Пользователь может удалить задачу. При этом удалённая задача пропадает из списка всех задач пользователя.
  • Пользователь может создать новую задачу.

О проекте

Это моя реализация одностраничного веб блога. Изначальная задумка была создать стартовую площадку для начинающих веб-разработчиков. Место, в котором можно черпать вдохновение и развиваться.

Технологии

Для создания блога использовались React, Django Rest Framework и, собственно, сам Django. В основном всё стандартно. На бэкенде имеются 2 основных приложения: 1 для постов и 1 для музыки. Фронт же состоит из нескольких компонентов. От отвечающего за рендер постов комопенета идет GET запрос и Django Rest Framework отдает данные в формате JSON обратно клиенту.
Код можно посмотреть здесь.

Features

  • Ссылка на запись блога имеет вид 'post/lorem-ipsum'. То есть вместо первичного ключа для формирования ссылки используется кастомное значение, которое можно задать в админке.
  • Посты на главной странице укорочены до 100 слов, после которых добавлена работающая кнопочка "Читать далее..".
  • Перед Футером рендарятся иконки-ссылки на мой инстаграм, телеграм и гитхаб.
  • В админке добавлена поддержка MarkDown разметки.
  • Добавлена возможность отложенной публикации.
  • Добавлена адаптивная панель навигации.
  • Записи упорядоченны по дате.
  • Добавлена пагинация.
