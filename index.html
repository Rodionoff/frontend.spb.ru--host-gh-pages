Немного о проекте

Это моя реализация одностраничного веб приложения. Изначальная задумка была создать стартовую площадку для начинающих веб-разработчиков. Место, в котором можно черпать вдохновение и развиваться.

Технологии

Для создания блога использовались React, Django Rest Framework и, собственно, сам Django. В основном всё стандартно. На бэкенде имеются 2 основных приложения: 1 для постов и 1 для музыки. Фронт же состоит из нескольких компонентов. От отвечающего за рендер постов комопенета идет GET запрос и Django Rest Framework отдает данные в формате JSON обратно клиенту. Код можно посмотреть здесь.

Features

  • Ссылка на запись имеет вид 'post/lorem-ipsum'. То есть вместо первичного ключа для формирования ссылки используются кастомное значение, которое можно задать в админке.
  • Посты на главной странице укорочены до 100 слов после которых добавлена работающаяя кнопочка "Читать далее..".
  • Перед Футером рендарятся иконки-ссылки на мой инстаграм, телеграм и гитхаб.
  • В админке добавлена поддержка MarkDown разметки.
  • Добавлена возможность отложенной публикации.
  • Добавлена адаптивная панель навигации.
  • Записи упорядоченны по дате.
  • Добавлена пагинация.
